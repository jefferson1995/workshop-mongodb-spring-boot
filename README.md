# Como executar o projeto

## Back end
Pré-requisitos: Java 11

```bash
# clonar repositório
git clone https://github.com/jefferson1995/workshop-springboot3-jpa.git

# entrar na pasta do projeto back end
cd workshop-springboot3-jpa


# executar o projeto
./mvnw spring-boot:run
```
